## ShopNow

ShopNow is an e-commerce website where a seller can sell their own products. A user can browse the site, view product information, and add items to the cart to purchase.

## Tech Stack

**Client:** React, next js, TailwindCSS,

**Server:** Node, Express js, mongobd

## Installation

Install my-project with npm

```bash
 git clone https://github.com/oiiemon9/shop-now.git
 cd shop-now
```

Install dependencies

```bash
 npm install
```

Create .env.local in the project root:

```bash
NEXT_PUBLIC_API_URL=your-api-url
DATABASE_URL=your-database-url
NEXTAUTH_SECRET=your-nextauth-secret
```

Run in development mode

```bash
npm run dev
```
